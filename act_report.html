Act_Report¶

KEY INSIGHTS¶

The data was derived from the twitter account: @rate_dogs.

The dataset used for the project is divided into three parts:

  1. Archived twitter data
  2. Image prediction data
  3. Retweet count and favourite count

Anaconda's Jupyter Notebook was majorly used for this project.

The wrangling efforts merged into the dateset into two datasets. From the main dataset that included the archived twitter data and the retweet count and favorite count data, using simple pandas functions like .value_counts(), I was able to determine that a large number of the dogs are without a name.

Having a name personalizes the experience for users who want to have their dogs rated. I recommend creating a format in which users are to drop the information. This in addition to personalizing the experience as previously stated also even allows easy data gathering and analysis to help promote the account.

Another insight gained from the Image prediction dataset is that most of the dogs are golden retrievers. Other dog species that are commonly seen are labrador and pembroke. Finally, most of the dog ratings were a 10/10.

VISUALIZATION¶

There are two visualizations displayed in this report. One is a wordcount that displays the name of the species based on their count. The other is a barplot that shows the average favorite count for each of the dog stages.

#  Using a word cloud to generate the p1 names of the dogs

names = ""
for i in cleaned_image['p1']:
    names = names + i + " "

# Create the wordcloud object
wordcloud = WordCloud(width=1000, height=1000).generate(names)
 
# Display the generated image:
plt.figure(figsize=(12, 7))
plt.imshow(wordcloud, interpolation='bilinear')
plt.axis("off")
plt.margins(x=0, y=0)

I quite like the art form of this visualization. It reveals that the golden retriever dogs are the most common species followed by the Labrador retriever and the Pembroke. Notable mentions also include the Chihuahua, toy poodle, Samoyed, Pug.
